Series Overview

The iOS rendering pipeline is one of the most frequently probed areas in Staff-level iOS interviews at Google, Apple, and Meta. Understanding why frames drop, where the process boundary sits between your app and the render server, and how the GPU composites layers is what separates senior engineers from architects.

This series takes you from the high-level Core Animation commit cycle all the way down to Metal compute shaders, covering every layer of the graphics stack that an iOS Staff engineer is expected to understand.

Architect Must-Know

At L6/Staff level, you're expected to not just know these systems but to design performance strategies around them. That means understanding the render server process boundary, knowing when off-screen rendering is unavoidable vs. fixable, and being able to architect image pipelines that handle thousands of assets without memory pressure.

Documents in This Series

01 — Core Animation Pipeline

Full render pipeline from app process commit transaction through the render server to GPU compositing. CATransaction, CALayer trees, ProMotion adaptive frame rates, and the four render loop phases.

02 — Off-Screen Rendering

What triggers off-screen rendering passes, how to measure the performance cost, and systematic approaches to eliminating every trigger. Instruments Core Animation template deep dive.

03 — Metal & GPU Programming

Metal fundamentals for iOS developers: MTKView, command buffers, render pipelines, compute shaders, Metal Performance Shaders, Core Image with Metal backend, and GPU frame capture.

04 — Text Rendering

TextKit 1 and TextKit 2 architectures, Core Text low-level API, custom text layout, emoji rendering complexity, Dynamic Type integration, and text performance for chat apps.

05 — Image Handling & Memory

UIImage memory cost analysis, ImageIO for efficient loading, downsampling techniques, HEIF/HEIC format, image caching architecture patterns, and building image pipelines for image-heavy apps.

06 — Scrolling Performance

The 60fps scroll checklist: cell preparation cost, background image decoding, prefetching APIs, shouldRasterize strategy, blended layers, misaligned images, and systematic jank diagnosis.

07 — Interview Questions & Answers

45+ L6-level interview questions with detailed answers covering the entire rendering and graphics stack. Tagged by difficulty with expandable answers.
