Architect-Level Focus Areas

Throughout the documentation, orange Architect callouts highlight knowledge that distinguishes a Staff/L6 engineer from a senior engineer. Key areas:

  • Flutter Engine internals — understanding Skia, Impeller, and how Flutter renders at 60/120fps (Series 1)
  • Three-tree architecture — Widget, Element, RenderObject relationships and lifecycle (Series 2)
  • State management trade-offs — when to use Provider vs Riverpod vs Bloc at scale (Series 3)
  • Navigator 2.0 declarative routing — RouterDelegate, RouteInformationParser for complex flows (Series 4)
  • Platform channel optimization — batching, threading, Pigeon code generation (Series 5)
  • Shader compilation jank — warmup strategies, Impeller benefits, profiling (Series 6)
  • CustomPainter and Canvas — efficient drawing, layer caching, repaint optimization (Series 7)
  • Golden testing strategies — pixel-perfect UI verification at scale (Series 8)
  • Secure storage architecture — Keychain/Keystore integration, encryption patterns (Series 9)
  • Shorebird OTA updates — code push for Flutter, rollback strategies (Series 10)
  • Isolate architecture — compute isolation, SendPort/ReceivePort patterns (Series 11)
  • Federated plugin architecture — platform interface patterns, Melos monorepo (Series 12)
