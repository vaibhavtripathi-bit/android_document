Key Themes for L6 Interviews

  • Navigator 2.0 components — RouterDelegate, RouteInformationParser, RouteInformationProvider roles
  • go vs push semantics — when each is appropriate and the UX bugs caused by using the wrong one
  • Auth guard pattern — redirect + refreshListenable + storing intended destination
  • StatefulShellRoute — per-tab back stacks vs ShellRoute's shared stack
  • Deep link cold vs warm start — GoRouter handles cold, stream subscription needed for warm
  • App Links over custom schemes — security, fallback to browser, no app chooser dialog
