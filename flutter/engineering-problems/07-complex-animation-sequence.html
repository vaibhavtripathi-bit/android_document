07 — Multi-Phase Animation Orchestration

Problem: Loading → success morph → text slide-in → dismiss sequence

Problem Statement

Build a payment confirmation widget with a four-phase animation that plays sequentially over 3 seconds using a single AnimationController:

0.0–0.3 Spinner
0.3–0.6 Morph
0.6–0.8 Text
0.8–1.0 Dismiss
  1. Phase 1 (0.0–0.3) — Circular spinner rotates continuously.
  2. Phase 2 (0.3–0.6) — Spinner arc shrinks while a checkmark path grows in its place.
  3. Phase 3 (0.6–0.8) — "Payment successful" text slides up from below with fade-in.
  4. Phase 4 (0.8–1.0) — Entire card fades out and translates downward to dismiss.
Staff Signal
The key design choice is using one AnimationController with Interval-based curves rather than chained controllers or Future.delayed sequencing. One controller means one source of truth for time, easy pause/resume, easy reverse, and trivial testing. Multiple controllers that call each other via addStatusListener are brittle and hard to test.

Architecture: Interval-Based Animations

CurvedAnimation with an Interval curve acts as a time gate. It outputs 0.0 when the controller value is before the interval start, ramps from 0.0 to 1.0 during the interval, and holds at 1.0 after the interval end. This lets N animations share one controller by selecting different time windows. 

// Conceptually:
// Interval(0.3, 0.6) maps controller value:
//   0.0 → 0.3   : outputs 0.0 (animation hasn't started)
//   0.3 → 0.6   : outputs 0.0 → 1.0 (animation running)
//   0.6 → 1.0   : outputs 1.0 (animation complete, held)

final morphProgress = Tween<double>(begin: 0, end: 1).animate(
  CurvedAnimation(
    parent: controller,
    curve: const Interval(0.3, 0.6, curve: Curves.easeInOut),
  ),
);

Complete Widget Implementation

State and Animation Setup

import 'dart:math';
import 'package:flutter/material.dart';

class PaymentSuccessWidget extends StatefulWidget {
  final VoidCallback? onDismissed;

  const PaymentSuccessWidget({super.key, this.onDismissed});

  @override
  State<PaymentSuccessWidget> createState() => _PaymentSuccessWidgetState();
}

class _PaymentSuccessWidgetState extends State<PaymentSuccessWidget>
    with SingleTickerProviderStateMixin {

  late AnimationController _controller;

  // Phase 1: 0.0–0.3 — spinner rotation (0 → 4π = 2 full rotations)
  late Animation<double> _spinnerRotation;

  // Phase 2: 0.3–0.6 — morph progress (0 = full spinner, 1 = full checkmark)
  late Animation<double> _morphProgress;

  // Phase 3: 0.6–0.8 — text Y offset (30px below → 0px = in place)
  late Animation<double> _textSlideY;
  late Animation<double> _textOpacity;

  // Phase 4: 0.8–1.0 — card opacity (1 → 0) and Y offset (0 → 40px down)
  late Animation<double> _dismissOpacity;
  late Animation<double> _dismissSlideY;

  @override
  void initState() {
    super.initState();

    _controller = AnimationController(
      duration: const Duration(milliseconds: 3000),
      vsync: this,
    );

    _spinnerRotation = Tween<double>(begin: 0, end: 4 * pi).animate(
      CurvedAnimation(
        parent: _controller,
        curve: const Interval(0.0, 0.3, curve: Curves.linear),
      ),
    );

    _morphProgress = Tween<double>(begin: 0, end: 1).animate(
      CurvedAnimation(
        parent: _controller,
        curve: const Interval(0.3, 0.6, curve: Curves.easeInOut),
      ),
    );

    _textSlideY = Tween<double>(begin: 30, end: 0).animate(
      CurvedAnimation(
        parent: _controller,
        curve: const Interval(0.6, 0.8, curve: Curves.easeOut),
      ),
    );

    _textOpacity = Tween<double>(begin: 0, end: 1).animate(
      CurvedAnimation(
        parent: _controller,
        curve: const Interval(0.6, 0.8, curve: Curves.easeOut),
      ),
    );

    _dismissOpacity = Tween<double>(begin: 1, end: 0).animate(
      CurvedAnimation(
        parent: _controller,
        curve: const Interval(0.8, 1.0, curve: Curves.easeIn),
      ),
    );

    _dismissSlideY = Tween<double>(begin: 0, end: 40).animate(
      CurvedAnimation(
        parent: _controller,
        curve: const Interval(0.8, 1.0, curve: Curves.easeIn),
      ),
    );

    _controller.addStatusListener((status) {
      if (status == AnimationStatus.completed) {
        widget.onDismissed?.call();
      }
    });

    _controller.forward();
  }

  @override
  void dispose() {
    _controller.dispose();
    super.dispose();
  }

  @override
  Widget build(BuildContext context) {
    return AnimatedBuilder(
      animation: _controller,
      builder: (context, _) {
        return Opacity(
          opacity: _dismissOpacity.value,
          child: Transform.translate(
            offset: Offset(0, _dismissSlideY.value),
            child: _buildCard(),
          ),
        );
      },
    );
  }

  Widget _buildCard() {
    return Container(
      width: 280,
      padding: const EdgeInsets.symmetric(vertical: 32, horizontal: 24),
      decoration: BoxDecoration(
        color: Colors.white,
        borderRadius: BorderRadius.circular(20),
        boxShadow: [
          BoxShadow(
            color: Colors.black.withOpacity(0.12),
            blurRadius: 24,
            offset: const Offset(0, 8),
          )
        ],
      ),
      child: Column(
        mainAxisSize: MainAxisSize.min,
        children: [
          // Spinner / checkmark morph area
          RepaintBoundary(
            child: CustomPaint(
              size: const Size(72, 72),
              painter: MorphPainter(
                spinnerRotation: _spinnerRotation.value,
                morphProgress: _morphProgress.value,
              ),
            ),
          ),
          const SizedBox(height: 24),
          // Sliding text — only visible in phase 3+
          Transform.translate(
            offset: Offset(0, _textSlideY.value),
            child: Opacity(
              opacity: _textOpacity.value,
              child: Column(
                children: const [
                  Text(
                    'Payment Successful',
                    style: TextStyle(
                      fontSize: 18,
                      fontWeight: FontWeight.w700,
                      color: Color(0xFF111827),
                    ),
                  ),
                  SizedBox(height: 6),
                  Text(
                    'Your order has been confirmed.',
                    style: TextStyle(fontSize: 14, color: Color(0xFF6B7280)),
                  ),
                ],
              ),
            ),
          ),
        ],
      ),
    );
  }
}

MorphPainter: Spinner → Checkmark

class MorphPainter extends CustomPainter {
  final double spinnerRotation; // Phase 1: 0 → 4π
  final double morphProgress;   // Phase 2: 0.0 → 1.0

  MorphPainter({
    required this.spinnerRotation,
    required this.morphProgress,
  });

  @override
  void paint(Canvas canvas, Size size) {
    final center = Offset(size.width / 2, size.height / 2);
    final radius = size.width / 2 - 6;

    // The spinner arc sweeps from full circle (2π) to 0 as morph progresses.
    final arcSweep = 2 * pi * (1 - morphProgress);

    // Arc color transitions from blue (spinner) to green (checkmark).
    final arcColor = Color.lerp(
      const Color(0xFF2563EB), // blue
      const Color(0xFF16A34A), // green
      morphProgress,
    )!;

    final arcPaint = Paint()
      ..color = arcColor
      ..style = PaintingStyle.stroke
      ..strokeWidth = 4.0
      ..strokeCap = StrokeCap.round;

    // Phase 1 + shrinking in phase 2: draw the rotating arc.
    if (arcSweep > 0.01) {
      canvas.drawArc(
        Rect.fromCircle(center: center, radius: radius),
        spinnerRotation - pi / 2, // start angle (rotate with controller)
        arcSweep,
        false,
        arcPaint,
      );
    }

    // Phase 2: draw checkmark growing via PathMetrics.
    if (morphProgress > 0) {
      _drawCheckmark(canvas, center, radius, morphProgress);
    }
  }

  void _drawCheckmark(
    Canvas canvas,
    Offset center,
    double radius,
    double progress,
  ) {
    // Checkmark path: two line segments forming a tick.
    // Scaled to fit inside the circle.
    final fullPath = Path()
      ..moveTo(center.dx - radius * 0.45, center.dy)
      ..lineTo(center.dx - radius * 0.1, center.dy + radius * 0.35)
      ..lineTo(center.dx + radius * 0.5, center.dy - radius * 0.35);

    // Use PathMetrics to extract only the portion we want to draw.
    final metric = fullPath.computeMetrics().first;
    final partial = metric.extractPath(0, metric.length * progress);

    final checkPaint = Paint()
      ..color = const Color(0xFF16A34A)
      ..style = PaintingStyle.stroke
      ..strokeWidth = 4.0
      ..strokeCap = StrokeCap.round
      ..strokeJoin = StrokeJoin.round;

    canvas.drawPath(partial, checkPaint);
  }

  // Only repaint when these values change.
  @override
  bool shouldRepaint(MorphPainter oldDelegate) {
    return oldDelegate.spinnerRotation != spinnerRotation ||
        oldDelegate.morphProgress != morphProgress;
  }
}

Key Design Decisions

Single Controller vs. Chained Controllers

ApproachProsCons
Single controller + Intervals (this solution)One timeline, easy pause/resume/reverse, easy testAll phases must share one total duration
Chained via addStatusListenerEach phase can have independent durationHard to pause mid-sequence, complex teardown, brittle
Future.delayed sequencingSimple to writeNot tied to vsync, can't pause, leaks if widget disposes early
TweenSequenceBuilt-in, good for value sequences on one animationOnly works for one animated value; can't orchestrate multiple independent properties

PathMetrics for Partial Path Drawing

PathMetrics is the correct way to draw a growing path. It computes the total arc length of any path (including curves) and lets you extract a sub-path from 0 to any fraction of that length. This works for any path — straight lines, Bezier curves, arcs — without needing to parameterize the path manually. 

// Pattern: draw first N% of a path
final metric = path.computeMetrics().first;
final partial = metric.extractPath(0, metric.length * progress);
canvas.drawPath(partial, paint);

Testing

void main() {
  testWidgets('MorphPainter shouldRepaint only on value change', (tester) async {
    final p1 = MorphPainter(spinnerRotation: 0, morphProgress: 0);
    final p2 = MorphPainter(spinnerRotation: 0, morphProgress: 0);
    final p3 = MorphPainter(spinnerRotation: 1, morphProgress: 0);

    expect(p1.shouldRepaint(p2), isFalse); // same values
    expect(p1.shouldRepaint(p3), isTrue);  // rotation changed
  });

  testWidgets('animation completes and fires onDismissed', (tester) async {
    bool dismissed = false;
    await tester.pumpWidget(
      MaterialApp(
        home: Scaffold(
          body: Center(
            child: PaymentSuccessWidget(
              onDismissed: () => dismissed = true,
            ),
          ),
        ),
      ),
    );

    // Advance to end of animation.
    await tester.pump();
    await tester.pump(const Duration(milliseconds: 3100));

    expect(dismissed, isTrue);
  });

  testWidgets('text is invisible at start, visible at phase 3', (tester) async {
    await tester.pumpWidget(
      const MaterialApp(
        home: Scaffold(body: Center(child: PaymentSuccessWidget())),
      ),
    );

    // At t=0, text opacity should be 0.
    await tester.pump();
    expect(
      find.text('Payment Successful'),
      findsOneWidget, // widget exists but opacity = 0
    );

    // Advance past phase 3 start (60% of 3000ms = 1800ms).
    await tester.pump(const Duration(milliseconds: 1900));
    // Text should now be partially visible — opacity > 0.
    // (Full assertion requires checking RenderOpacity.opacity via finder)
  });
}

Accessibility Considerations

  • Wrap the entire widget in a Semantics node with label: 'Payment successful' so screen readers announce the outcome immediately without waiting for the animation.
  • Respect MediaQuery.of(context).disableAnimations — if true, skip to the final state immediately via _controller.value = 1.0.
  • The dismiss phase should call onDismissed from addStatusListener, not from a timer, so callers can navigate away cleanly.
@override
void initState() {
  super.initState();
  // ... setup animations ...

  // Skip animation for users who prefer reduced motion.
  WidgetsBinding.instance.addPostFrameCallback((_) {
    if (MediaQuery.of(context).disableAnimations) {
      _controller.value = 0.79; // jump to just before dismiss phase
    } else {
      _controller.forward();
    }
  });
}
