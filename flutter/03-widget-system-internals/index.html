Key Concepts to Know Cold

  • Widget.canUpdate(old, new) — runtimeType AND key must match for element reuse
  • Object identity short-circuit — const widgets skip reconciliation via identical() check in updateChild()
  • _inheritedElements map — O(1) InheritedWidget lookup, propagated during mount, copied on InheritedElement insertion
  • _dependents / _dependencies — bidirectional registration between InheritedElement and dependent elements
  • Two-pass reconciliation — linear positional scan then keyed HashMap match for multi-child lists
  • GlobalKey inactive pool — one-frame window for element portability across tree locations
  • Build scope ordering — dirty elements rebuilt shallow-first to prevent double rebuilds
  • context.mounted — Flutter 3.0+ guard against async use-after-unmount
