React Native Skia

@shopify/react-native-skia brings Skia — the 2D graphics engine behind Chrome, Android, and Flutter — to React Native. It enables custom drawing, image filters, shaders, blur effects, and high-performance rendering that goes beyond what the native view system can achieve.

1. Skia Architecture

React Native Skia Architecture ════════════════════════════════════════════════════ React Component Tree │ ▼ ┌─────────────────────────────────────────┐ │ <Canvas> Component │ │ ┌─────────────────────────────────┐ │ │ │ Skia Declarative API │ │ │ │ <Rect>, <Circle>, <Path>, │ │ │ │ <Image>, <Text>, <Group> │ │ │ └──────────────┬──────────────────┘ │ │ │ │ │ ▼ │ │ ┌─────────────────────────────────┐ │ │ │ Skia Render Tree │ │ │ │ (C++ objects via JSI) │ │ │ └──────────────┬──────────────────┘ │ │ │ │ │ ▼ │ │ ┌─────────────────────────────────┐ │ │ │ Skia C++ Engine │ │ │ │ • GPU-accelerated rendering │ │ │ │ • Same engine as Chrome/Flutter│ │ │ │ • Hardware-accelerated shaders │ │ │ └──────────────┬──────────────────┘ │ │ │ │ │ ▼ │ │ ┌─────────────────────────────────┐ │ │ │ Native Surface │ │ │ │ iOS: Metal / Android: OpenGL │ │ │ └─────────────────────────────────┘ │ └─────────────────────────────────────────┘ Key: Skia renders to its own surface, independent of the React Native view hierarchy. It's a canvas, not views.
Key Insight

React Native Skia uses a declarative API that mirrors React's component model. Instead of imperative canvas drawing (like HTML5 Canvas), you compose Skia primitives as JSX elements inside a <Canvas>. The Skia render tree is separate from RN's view tree — it renders to its own GPU surface. This means Skia drawing doesn't interfere with RN's layout system, but also means you can't mix Skia elements with native views inside the same canvas.

      {/* Oval */}
      <Oval
        x={20} y={140} width={150} height={80}

      />

      {/* Line */}
      <Line
        p1={vec(20, 250)} p2={vec(280, 250)}
        color="#E74C3C"
        strokeWidth={3}
        style="stroke"
      />

      {/* Text */}
      {font && (
        <Text
          x={20} y={300}
          text="Hello Skia!"
          font={font}
          color="#2C3E50"
        />
      )}

      {/* Image */}
      {image && (
        <Image
          image={image}
          x={20} y={320}
          width={200} height={150}
          fit="cover"
        />
      )}
    </Canvas>
  );
}

4. Paths & Shapes

import { Canvas, Path, Skia } from '@shopify/react-native-skia';

// SVG path string
<Path
  path="M 20 60 Q 75 10 130 60 Q 185 110 240 60"
  color="#E74C3C"
  style="stroke"
  strokeWidth={3}
  strokeCap="round"
/>

// Programmatic path construction
function WaveChart({ data }: { data: number[] }) {
  const path = useMemo(() => {
    const p = Skia.Path.Make();
    const stepX = 300 / (data.length - 1);

    p.moveTo(0, 200 - data[0] * 2);
    for (let i = 1; i < data.length; i++) {
      const x = i * stepX;
      const y = 200 - data[i] * 2;
      const prevX = (i - 1) * stepX;
      const prevY = 200 - data[i - 1] * 2;
      const cpX = (prevX + x) / 2;
      p.cubicTo(cpX, prevY, cpX, y, x, y);
    }

    return p;
  }, [data]);

  return (
    <Canvas style={{ width: 300, height: 200 }}>
      <Path
        path={path}
        color="#4A90D9"
        style="stroke"
        strokeWidth={2}
        strokeCap="round"
        strokeJoin="round"
      />
    </Canvas>
  );
}

// Gradient fills
<Rect x={0} y={0} width={300} height={200}>
  <LinearGradient
    start={vec(0, 0)}
    end={vec(300, 200)}
    colors={['#4A90D9', '#9B59B6', '#E74C3C']}
  />
</Rect>

// Custom GLSL shader
const shaderSource = Skia.RuntimeEffect.Make(`
  uniform float time;
  uniform float2 resolution;

  half4 main(float2 pos) {
    float2 uv = pos / resolution;
    float wave = sin(uv.x * 10.0 + time) * 0.5 + 0.5;
    return half4(uv.x, wave, uv.y, 1.0);
  }
`);

function ShaderAnimation() {
  const time = useSharedValue(0);

  useEffect(() => {
    time.value = withRepeat(
      withTiming(Math.PI * 2, { duration: 3000, easing: Easing.linear }),
      -1, false
    );
  }, []);

  const uniforms = useDerivedValue(() => ({
    time: time.value,
    resolution: [300, 300],
  }));

  return (
    <Canvas style={{ width: 300, height: 300 }}>
      <Fill>
        <Shader source={shaderSource} uniforms={uniforms} />
      </Fill>
    </Canvas>
  );
}
Architect Insight

Skia shaders run on the GPU, making them extremely performant for visual effects. A shader that would be impossible to achieve with the view system (animated gradients, wave effects, blur transitions) runs at 60fps with minimal CPU overhead. However, shaders are written in GLSL (or Skia's SkSL variant) — a different programming model that requires GPU programming knowledge. Use shaders for effects that can't be achieved with Reanimated + views, not as a replacement for standard UI.

6. Reanimated Integration

import { Canvas, Circle, Group } from '@shopify/react-native-skia';
import { useSharedValue, useDerivedValue, withRepeat, withTiming } from 'react-native-reanimated';

function AnimatedSkia() {
  const progress = useSharedValue(0);

  useEffect(() => {
    progress.value = withRepeat(
      withTiming(1, { duration: 2000 }),
      -1, true
    );
  }, []);

  const cx = useDerivedValue(() => 50 + progress.value * 200);
  const r = useDerivedValue(() => 20 + progress.value * 30);
  const opacity = useDerivedValue(() => 0.3 + progress.value * 0.7);

  return (
    <Canvas style={{ width: 300, height: 200 }}>
      <Group opacity={opacity}>
        <Circle cx={cx} cy={100} r={r} color="#4A90D9" />
      </Group>
    </Canvas>
  );
}

// Gesture-driven Skia drawing
function DrawingCanvas() {
  const [paths, setPaths] = useState<SkPath[]>([]);
  const currentPath = useSharedValue<SkPath | null>(null);

  const pan = Gesture.Pan()
    .onStart((e) => {
      const path = Skia.Path.Make();
      path.moveTo(e.x, e.y);
      currentPath.value = path;
    })
    .onUpdate((e) => {
      currentPath.value?.lineTo(e.x, e.y);
    })
    .onEnd(() => {
      if (currentPath.value) {
        runOnJS(setPaths)([...paths, currentPath.value]);
        currentPath.value = null;
      }
    });

  return (
    <GestureDetector gesture={pan}>
      <Canvas style={{ flex: 1 }}>
        {paths.map((path, i) => (
          <Path key={i} path={path} color="black" style="stroke" strokeWidth={3} />
        ))}
      </Canvas>
    </GestureDetector>
  );
}

8. Architect Considerations

Staff-Level Mental Model

React Native Skia fills the gap between the view system and native custom drawing. The view system (RN components) handles layout, text, and standard UI. Reanimated handles animation of view properties. Skia handles everything else: custom shapes, GPU shaders, image processing, blur effects, and complex visual compositions. A Staff engineer reaches for Skia when the requirement exceeds what views + transforms can achieve — custom charts, frosted glass, drawing canvases, particle effects. The key trade-off: Skia renders to its own surface (not the view hierarchy), so you can't put native views inside a Skia canvas or use accessibility APIs on Skia-drawn elements. Use Skia for visual effects and custom rendering; use views for interactive, accessible UI.
