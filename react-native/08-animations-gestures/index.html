Series 8: Animations & Gestures

From the Animated API to Reanimated worklets, Gesture Handler, Skia graphics, and shared element transitions — the complete animation stack for 60fps mobile experiences.

Animations and gestures are what separate a mediocre mobile app from a delightful one. React Native provides a layered animation system: the built-in Animated API for simple cases, Reanimated v3 for UI-thread worklets, Gesture Handler for native gesture recognition, and Skia for custom 2D graphics. This series dissects each layer at the depth expected for Google L6 interviews — from how useNativeDriver serializes animation configs to the Reanimated worklet VM running on the UI thread, from gesture composition APIs to Skia shader programs.

Documents in This Series

01 — Animated API

Animated.Value, timing/spring/decay, composition (sequence, parallel, stagger), interpolation, Animated.event, useNativeDriver internals, and Animated.ValueXY for 2D animations.

02 — Reanimated Internals

Reanimated v3 worklet runtime, shared values, useAnimatedStyle, worklet threading model, layout animations, JSI integration, and the UI-thread JS VM architecture.

03 — Gesture Handler

Native gesture recognition, GestureDetector composable API, Pan/Pinch/Rotation/Tap/LongPress/Fling, gesture composition, platform differences, and Reanimated integration.

04 — Layout Animations

Entering/exiting animations, layout transitions, keyframe animations, FlatList animation, LayoutAnimation legacy API, custom animations, and performance considerations.

05 — Shared Element Transitions

Cross-screen shared elements, navigation integration, Reanimated shared transitions, hero animations, iOS vs Android differences, and custom page transitions.

06 — Advanced Patterns

Physics-based animations, gesture-driven interactions (swipeable cards, bottom sheets), skeleton loading, Lottie integration, SVG animations, particle systems, and performance profiling.

07 — Skia Graphics

React Native Skia: 2D rendering, drawing primitives, image filters, shaders, blur effects, gradients, Reanimated integration, and custom chart/UI rendering.

08 — Interview Q&A

55+ L6-level questions covering Animated API, Reanimated internals, Gesture Handler, layout animations, shared elements, Skia, and advanced animation patterns.
