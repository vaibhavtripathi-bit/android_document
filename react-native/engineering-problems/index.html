Engineering Problems

System design and architecture problems for Staff-level mobile engineering interviews. Each problem covers end-to-end design — from requirements gathering and architecture diagrams through implementation details, trade-off analysis, and production hardening. These are the open-ended problems that separate Senior from Staff engineers.

Staff-level interviews increasingly focus on engineering problem-solving rather than algorithmic puzzles. You'll be asked to design, build, and scale real mobile systems — chat apps, offline-first architectures, migration strategies, and more. Each problem below is a deep dive into one such system, covering the architecture decisions, trade-offs, and production concerns that interviewers expect at L6+.

01

Chat App Architecture

Real-time messaging with WebSocket management, message delivery guarantees, offline queuing, read receipts, typing indicators, push notifications, and file sharing.

 02

Offline-First Mobile App

Local database design (SQLite/Realm), sync engine architecture, conflict resolution strategies, optimistic UI, background sync, and data migration.

 03

App Performance Audit

Startup time analysis, frame rate profiling, memory leak detection, bundle size analysis, network waterfall, performance budgets, and regression testing.

 04

Native → React Native Migration

Brownfield integration, shared navigation, incremental migration plan, feature parity assessment, performance benchmarking, team training, and rollback strategy.

 05

Cross-Platform Design System

Token architecture, component API design, theming engine, platform-adaptive components, accessibility, documentation, and versioning strategy.

 06

Push Notification System

APNs/FCM integration, notification channels, deep linking, rich notifications, silent pushes, delivery tracking, and notification preferences.

 07

OTA Update System

CodePush architecture, update strategies, rollback mechanisms, A/B testing, binary vs JS updates, version compatibility, and deployment pipelines.

 08

Analytics Pipeline

Event tracking architecture, batching and compression, offline buffering, schema validation, privacy compliance, real-time dashboards, and A/B test integration.

 09

Authentication Flow

OAuth/OIDC integration, biometric auth, token management, session handling, multi-device sync, secure storage, and account recovery flows.

 10

Media Player Architecture

Audio/video playback, streaming protocols, DRM, background playback, PiP, casting, download management, and adaptive bitrate streaming.

 11

E-Commerce App

Product catalog, cart management, checkout flow, payment integration, inventory sync, search and filtering, and order tracking.

 12

Background Processing

Background tasks on iOS/Android, WorkManager/BGTaskScheduler, upload queues, geofencing, periodic sync, battery optimization, and headless JS.
