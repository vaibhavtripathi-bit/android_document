Architect-Level Focus Areas

Throughout the documentation, orange Architect callouts highlight knowledge that distinguishes a Staff/L6 engineer from a senior engineer. Key areas:

  • Bridge serialization bottleneck — understanding why the old architecture is slow and how JSI solves it (Series 1)
  • Fabric C++ renderer — ShadowNode tree, concurrent rendering, synchronous layout (Series 2)
  • View flattening — how React Native optimizes the native view hierarchy (Series 3)
  • Native stack internals — UINavigationController/Fragment integration, screen lifecycle (Series 4)
  • TurboModule codegen — type-safe native interfaces, spec files, build integration (Series 5)
  • 60fps performance — JS thread budget, native driver, Reanimated worklets (Series 6)
  • Persistent storage architecture — MMKV vs SQLite vs Realm, encryption, migration (Series 7)
  • Reanimated worklet runtime — separate JS VM on UI thread, shared values (Series 8)
  • Universal app architecture — React Native for Web, Solito, Tamagui (Series 9)
  • Gray-box E2E testing — Detox synchronization, native module testing (Series 10)
  • Certificate pinning — TLS security, Keychain/Keystore, code protection (Series 11)
  • OTA update architecture — CodePush/EAS Update, delta updates, rollback (Series 12)
