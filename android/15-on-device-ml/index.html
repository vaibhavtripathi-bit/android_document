On-Device ML

TFLite · ML Kit · Model Optimization · GPU/NNAPI Delegates · Inference Pipeline — L6 Interview Prep

Why this matters at L6 On-device ML is the differentiator for Staff-level candidates. Companies like Google, Samsung, and Flipkart are shipping ML features (smart search, visual search, content recommendations, text recognition) that run entirely on the device — no network round-trip, works offline, privacy-preserving. L6 design rounds increasingly include "how would you run this model on-device efficiently?" questions.
