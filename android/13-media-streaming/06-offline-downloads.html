Offline Downloads (DRM-Protected)

DownloadManager, DownloadRequest, offline DRM key pre-fetching, storage, resume/retry — the Netflix & Hotstar "Download" feature

1. What "Download for Offline" Actually Means

When a user taps "Download" on Netflix or Hotstar, four things must happen:

  1. Segment download — all video, audio, and subtitle segments are downloaded to local storage.
  2. Manifest storage — the HLS/DASH manifest is saved so the player can reconstruct the stream locally.
  3. DRM key pre-fetch — if the content is DRM-protected, an offline license (containing the decryption key) must be obtained from the license server before the user goes offline.
  4. Playback routing — when the user plays, ExoPlayer reads from local files instead of the network.
DRM Offline Is the Hard Part
Downloading plain (non-DRM) content is straightforward. DRM-protected offline is where most complexity lies — offline licenses have expiry dates, must be renewed, and can be revoked server-side.

2. The ExoPlayer Download Architecture

Your App │ ├── DownloadManager ← orchestrates all downloads │ ├── DownloadRequest ← what to download (URI, quality, DRM config) │ ├── DownloadIndex ← persists download state (SQLite via StandaloneDatabaseProvider) │ └── Downloader ← does the actual segment fetching │ ├── ProgressiveDownloader (single-file, non-adaptive) │ ├── HlsDownloader (HLS segments) │ └── DashDownloader (DASH segments) │ ├── DownloadService ← ForegroundService that survives app backgrounding │ └── runs DownloadManager in foreground │ └── Cache (SimpleCache) ← local storage, shared between download + playback └── CacheDataSource ← DataSource that reads from Cache first, network second

3. Setting Up the Download Infrastructure

// DownloadUtil.kt — singleton helpers (create once, reuse)
object DownloadUtil {

    private lateinit var downloadManager: DownloadManager
    private lateinit var downloadCache: SimpleCache

    fun getDownloadManager(context: Context): DownloadManager {
        if (!::downloadManager.isInitialized) {
            val databaseProvider = StandaloneDatabaseProvider(context)
            downloadCache = SimpleCache(
                File(context.filesDir, "downloads"),  // internal storage
                NoOpCacheEvictor(),                   // never evict — user manages storage
                databaseProvider
            )
            val dataSourceFactory = DefaultHttpDataSource.Factory()
            downloadManager = DownloadManager(
                context,
                databaseProvider,
                downloadCache,
                dataSourceFactory,
                Executor(Runnable::run)               // or use a thread pool
            ).apply {
                maxParallelDownloads = 3              // 3 concurrent downloads
            }
        }
        return downloadManager
    }

    fun getDownloadCache(context: Context): SimpleCache {
        getDownloadManager(context)  // ensures cache is initialized
        return downloadCache
    }
}

4. Starting a Download

// Download a DASH stream at a specific quality
fun downloadContent(context: Context, contentId: String, streamUrl: String, token: String) {

    // 1. Build MediaItem with DRM config (same as streaming)
    val mediaItem = MediaItem.Builder()
        .setUri(streamUrl)
        .setDrmConfiguration(
            MediaItem.DrmConfiguration.Builder(C.WIDEVINE_UUID)
                .setLicenseUri("https://license.hotstar.com/widevine")
                .setLicenseRequestHeaders(mapOf("Authorization" to "Bearer $token"))
                .build()
        )
        .build()

    // 2. Build DownloadRequest — specify which tracks to download
    val downloadRequest = DownloadRequest.Builder(contentId, mediaItem.localConfiguration!!.uri)
        .setMimeType(MimeTypes.APPLICATION_MPD)   // DASH
        // Optionally restrict to specific track (e.g., 720p video + Hindi audio)
        .setKeySetId(null)   // will be populated after DRM key fetch
        .build()

    // 3. Send to DownloadService
    DownloadService.sendAddDownload(
        context,
        VideoDownloadService::class.java,
        downloadRequest,
        /* foreground= */ false
    )
}
L6 Insight — Track Selection for Downloads
By default, ExoPlayer's downloader downloads ALL tracks at ALL quality levels — potentially many GB. In production, you must specify which tracks to download. Use DownloadHelper to inspect available tracks and let the user choose quality (or pick a smart default like 720p + highest bitrate audio): 
// Use DownloadHelper to pick specific tracks before downloading
val downloadHelper = DownloadHelper.forMediaItem(
    context,
    mediaItem,
    DefaultRenderersFactory(context),
    DefaultTrackSelector(context).also { selector ->
        // Restrict to 720p video + Hindi audio
        selector.parameters = selector.buildUponParameters()
            .setMaxVideoSize(1280, 720)
            .setPreferredAudioLanguage("hi")
            .build()
    }
)

downloadHelper.prepare(object : DownloadHelper.Callback {
    override fun onPrepared(helper: DownloadHelper) {
        // helper.getTrackGroups(0) shows available tracks
        val downloadRequest = helper.getDownloadRequest(
            contentId,
            /* data= */ null
        )
        DownloadService.sendAddDownload(
            context, VideoDownloadService::class.java, downloadRequest, false
        )
        helper.release()
    }

    override fun onPrepareError(helper: DownloadHelper, e: IOException) {
        helper.release()
        showError("Could not prepare download: ${e.message}")
    }
})

5. Offline DRM License — The Critical Step

For DRM-protected content, an offline license must be pre-fetched before the user loses connectivity. An offline license is a license where the key is stored on-device (in the TEE for L1, or encrypted on disk for L3) — it does not require a network call during playback.

Online (during download): 1. DownloadManager fetches segments (encrypted) 2. App requests offline license from license server POST /widevine-license body: { challenge, contentId, userId } header: Authorization: Bearer token 3. License server responds with offline license (includes: content key wrapped for device, + rental_duration_seconds) 4. Widevine CDM stores key in TEE / encrypted storage 5. keySetId saved in DownloadIndex (maps content → key) Offline (during playback, no network): 1. ExoPlayer reads local segments from SimpleCache 2. Passes keySetId to OfflineLicenseHelper 3. Widevine CDM retrieves stored key — NO NETWORK CALL 4. Segments decrypted and played normally 
// Pre-fetch DRM offline license during download
suspend fun fetchOfflineLicense(
    context: Context,
    mediaItem: MediaItem,
    contentId: String
): ByteArray {
    val offlineLicenseHelper = OfflineLicenseHelper.newWidevineInstance(
        "https://license.hotstar.com/widevine",
        true,  // forceDefaultLicenseUri
        DefaultHttpDataSource.Factory().apply {
            setDefaultRequestProperties(mapOf("Authorization" to "Bearer $userToken"))
        },
        DrmSessionEventListener.EventDispatcher()
    )

    return withContext(Dispatchers.IO) {
        try {
            val keySetId = offlineLicenseHelper.downloadLicense(mediaItem)
            // keySetId is a byte array — save it, associate with contentId
            saveKeySetId(contentId, keySetId)
            keySetId
        } finally {
            offlineLicenseHelper.release()
        }
    }
}

// Check license expiry
suspend fun checkLicenseExpiry(context: Context, keySetId: ByteArray): Pair<Long, Long> {
    val helper = OfflineLicenseHelper.newWidevineInstance(...)
    return try {
        helper.getLicenseDurationRemainingSec(keySetId)
        // Returns Pair(licenseDurationRemainingSec, playbackDurationRemainingSec)
    } finally {
        helper.release()
    }
}

// Renew before expiry
suspend fun renewLicense(context: Context, keySetId: ByteArray): ByteArray {
    val helper = OfflineLicenseHelper.newWidevineInstance(...)
    return try {
        helper.renewLicense(keySetId)
    } finally {
        helper.release()
    }
}

6. Playing Downloaded Content

// Build a DataSource that reads from cache first, network second
fun buildCacheDataSourceFactory(context: Context): CacheDataSource.Factory {
    val cache = DownloadUtil.getDownloadCache(context)
    val upstreamFactory = DefaultHttpDataSource.Factory()
    return CacheDataSource.Factory()
        .setCache(cache)
        .setUpstreamDataSourceFactory(upstreamFactory)
        .setCacheWriteDataSinkFactory(null)  // don't write to cache during playback
        .setFlags(CacheDataSource.FLAG_IGNORE_CACHE_ON_ERROR)
}

// Build player for offline playback
fun buildOfflinePlayer(context: Context, contentId: String): ExoPlayer {
    val keySetId = loadKeySetId(contentId)  // retrieve saved DRM key ID

    val drmConfig = MediaItem.DrmConfiguration.Builder(C.WIDEVINE_UUID)
        .setLicenseUri("https://license.hotstar.com/widevine")
        // For offline: use the stored keySetId instead of fetching new license
        .setKeySetId(keySetId)
        .build()

    val mediaItem = MediaItem.Builder()
        .setUri(getLocalManifestUri(contentId))   // local file URI
        .setDrmConfiguration(drmConfig)
        .build()

    return ExoPlayer.Builder(context)
        .setMediaSourceFactory(
            DefaultMediaSourceFactory(context)
                .setDataSourceFactory(buildCacheDataSourceFactory(context))
        )
        .build().also { player ->
            player.setMediaItem(mediaItem)
            player.prepare()
        }
}

7. Download State Machine

// Observe all downloads and their states
val downloadManager = DownloadUtil.getDownloadManager(context)

downloadManager.addListener(object : DownloadManager.Listener {
    override fun onDownloadChanged(
        downloadManager: DownloadManager,
        download: Download,
        finalException: Exception?
    ) {
        when (download.state) {
            Download.STATE_QUEUED      -> updateUI("Queued")
            Download.STATE_DOWNLOADING -> {
                val progress = download.percentDownloaded
                updateUI("Downloading: ${progress.toInt()}%")
            }
            Download.STATE_COMPLETED   -> updateUI("Ready to watch offline")
            Download.STATE_FAILED      -> updateUI("Failed: ${finalException?.message}")
            Download.STATE_REMOVING    -> updateUI("Removing...")
            Download.STATE_STOPPED     -> updateUI("Paused")
            Download.STATE_RESTARTING  -> updateUI("Restarting...")
        }
    }
})

// Pause/resume/remove
fun pauseDownload(contentId: String) {
    DownloadService.sendSetStopReason(
        context, VideoDownloadService::class.java,
        contentId, Download.STOP_REASON_APP, false
    )
}

fun resumeDownload(contentId: String) {
    DownloadService.sendSetStopReason(
        context, VideoDownloadService::class.java,
        contentId, Download.STOP_REASON_NONE, false
    )
}

fun deleteDownload(contentId: String) {
    DownloadService.sendRemoveDownload(
        context, VideoDownloadService::class.java, contentId, false
    )
}

8. Storage Management

ConcernApproach
Where to storeInternal storage (context.filesDir) for DRM content — DRM keys are bound to device, so external SD card causes issues across devices. Use getExternalFilesDir() only for non-DRM content.
Storage check before downloadUse StatFs to check available bytes before queuing. Reject if available < estimated download size + 500MB buffer.
Eviction policyNoOpCacheEvictor for downloads (never auto-evict). User explicitly deletes. LRU eviction is suitable only for streaming cache.
Cache vs downloadUse one shared SimpleCache. Downloads write to it. Streaming playback reads from it first, then network. This means a streamed video is partially cached and can be completed as a download.
License expiry notificationUse WorkManager periodic task to check getLicenseDurationRemainingSec(). Notify user to open app (to renew) 48h before expiry.

Interview Cheat Sheet — Offline Downloads

  • Four things "Download" does? — Segment download, manifest storage, DRM offline license pre-fetch, playback routing to cache.
  • Key classes? — DownloadManager (orchestrates), DownloadService (ForegroundService), SimpleCache (storage), OfflineLicenseHelper (DRM key pre-fetch), CacheDataSource (cache-first playback).
  • What is a keySetId? — A byte array returned by the CDM after fetching an offline license. Save it (associated with contentId) and pass it to DrmConfiguration during offline playback so the CDM retrieves the stored key without a network call.
  • DRM + external storage? — Avoid. DRM keys are device-bound. If you store segments on an SD card that's moved to another device, playback fails. Use internal storage for DRM content.
  • How to pick quality for download? — Use DownloadHelper to enumerate tracks, then configure a TrackSelector with desired constraints (e.g., 720p, Hindi audio). Pass result to getDownloadRequest().
  • License expiry handling? — Use OfflineLicenseHelper.getLicenseDurationRemainingSec(). Renew via renewLicense(keySetId) before expiry. Schedule a WorkManager check 48h before expiry.
  • Download resumability? — ExoPlayer's DownloadManager stores download state in SQLite via DownloadIndex. On app restart/service restart, downloads resume from last completed segment.
  • Shared cache strategy? — One SimpleCache shared between downloading and streaming. A partially-streamed video is already in cache — completing it as a download only fetches the missing segments.
