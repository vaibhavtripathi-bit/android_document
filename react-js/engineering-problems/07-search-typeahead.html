Building Search Typeahead at Scale

Debouncing, cancellation, caching, keyboard UX, accessibility, highlighting, recents, trending, and rate limits—end to end.

Staff takeaway

Typeahead is a distributed systems problem in miniature: stale responses, thundering herds, and accessibility state machines all collide in one input. The production solution is explicit about ordering, cancellation, and announcements.

Debouncing vs throttling

Debounce waits for a pause in input; ideal when each keystroke would otherwise trigger an expensive query. Throttle enforces a maximum rate regardless of burstiness; better for scroll or resize, rarely primary for search unless you cap server QPS.

DEBOUNCE (wait 300ms after last key) Keys: r ─ e ─ a ─ c ─ t │ │ │ │ └─── fire ──► request("react") └───┴───┴───┴──────── (earlier chars never fire) THROTTLE (max 1 call / 300ms wall clock) Keys: r ─ e ─ a ─ c ─ t ▼ ▼ ▼ req("r") skip skip req("react") ← may skip intermediate states
Technique Server load Perceived latency Stale risk
Debounce 200–400ms Low Slight delay after typing stops Lower (fewer in-flight)
Throttle Bounded More intermediate results Higher cancellation need
Leading debounce Medium Snappier first char Needs merge with trailing
Hybrid pattern

Use a short leading request for the first character (instant feedback) and debounce subsequent edits, or debounce + prefetch on focus for empty-query defaults.

Request cancellation with AbortController

Network latency is non-deterministic: response for "rea" may arrive after "react". Abort previous fetches when the query changes; ignore payloads that match an aborted signal.

LRU cache for queries

Cache normalized query strings to keys; cap entries (e.g., 100) and evict LRU. Store both results and negative cache entries for empty results to avoid hammering the backend.

PII in localStorage

Recent searches may contain health, financial, or personal strings. Offer clear-user-data, encrypt at rest if required, or keep recents server-side with consent.

Accessibility: ARIA combobox

Follow the WAI-ARIA combobox pattern: input has role="combobox", aria-expanded, aria-controls pointing to listbox, aria-activedescendant for virtual focus. Announce result counts with a polite live region.

Production typeahead component (TypeScript + React)

// Typeahead.tsx — single-file production-shaped example
import {
  useCallback,
  useEffect,
  useId,
  useMemo,
  useRef,
  useState,
  type KeyboardEvent,
  type ChangeEvent,
} from 'react';

type Suggestion = { id: string; label: string };

const RECENTS_KEY = 'typeahead:recents:v1';
const MAX_RECENTS = 8;
const CACHE_CAP = 128;

class LruCache<K, V> {
  private map = new Map<K, V>();
  constructor(private cap: number) {}
  get(k: K) {
    if (!this.map.has(k)) return undefined;
    const v = this.map.get(k)!;
    this.map.delete(k);
    this.map.set(k, v);
    return v;
  }
  set(k: K, v: V) {
    if (this.map.has(k)) this.map.delete(k);
    this.map.set(k, v);
    if (this.map.size > this.cap) {
      const first = this.map.keys().next().value;
      this.map.delete(first);
    }
  }
}

function normalize(q: string) {
  return q.trim().toLowerCase().replace(/\s+/g, ' ');
}

function loadRecents(): string[] {
  try {
    const raw = localStorage.getItem(RECENTS_KEY);
    return raw ? (JSON.parse(raw) as string[]) : [];
  } catch {
    return [];
  }
}

function saveRecents(items: string[]) {
  localStorage.setItem(RECENTS_KEY, JSON.stringify(items.slice(0, MAX_RECENTS)));
}

function highlight(label: string, q: string) {
  const n = normalize(q);
  if (!n) return label;
  const idx = label.toLowerCase().indexOf(n);
  if (idx < 0) return label;
  const before = label.slice(0, idx);
  const match = label.slice(idx, idx + n.length);
  const after = label.slice(idx + n.length);
  return (
    <>
      {before}
      <strong>{match}</strong>
      {after}
    </>
  );
}

async function fetchSuggestions(query: string, signal: AbortSignal): Promise<Suggestion[]> {
  const url = `/api/search/suggest?q=${encodeURIComponent(query)}`;
  const res = await fetch(url, { signal, headers: { Accept: 'application/json' } });
  if (!res.ok) throw new Error(String(res.status));
  return res.json();
}

export function Typeahead({
  trending,
}: {
  trending: Suggestion[];
}) {
  const listId = useId();
  const statusId = useId();
  const [value, setValue] = useState('');
  const [open, setOpen] = useState(false);
  const [activeIdx, setActiveIdx] = useState(-1);
  const [loading, setLoading] = useState(false);
  const [items, setItems] = useState<Suggestion[]>([]);
  const [announce, setAnnounce] = useState('');

  const cache = useMemo(() => new LruCache<string, Suggestion[]>(CACHE_CAP), []);
  const abortRef = useRef<AbortController | null>(null);
  const debounceRef = useRef<number | null>(null);
  const seqRef = useRef(0);

  const runQuery = useCallback(
    async (q: string) => {
      const key = normalize(q);
      if (!key) {
        setItems(trending);
        setAnnounce(`${trending.length} trending suggestions`);
        return;
      }
      const hit = cache.get(key);
      if (hit) {
        setItems(hit);
        setAnnounce(`${hit.length} results for ${q}`);
        return;
      }
      abortRef.current?.abort();
      const ctrl = new AbortController();
      abortRef.current = ctrl;
      const mySeq = ++seqRef.current;
      setLoading(true);
      try {
        const data = await fetchSuggestions(key, ctrl.signal);
        if (mySeq !== seqRef.current) return;
        cache.set(key, data);
        setItems(data);
        setAnnounce(`${data.length} results for ${q}`);
      } catch (e) {
        if ((e as Error).name === 'AbortError') return;
        setItems([]);
        setAnnounce('Search failed');
      } finally {
        if (mySeq === seqRef.current) setLoading(false);
      }
    },
    [cache, trending],
  );

  const schedule = useCallback(
    (q: string) => {
      if (debounceRef.current) window.clearTimeout(debounceRef.current);
      debounceRef.current = window.setTimeout(() => runQuery(q), 280);
    },
    [runQuery],
  );

  useEffect(() => () => {
    if (debounceRef.current) window.clearTimeout(debounceRef.current);
    abortRef.current?.abort();
  }, []);

  function onChange(e: ChangeEvent<HTMLInputElement>) {
    const q = e.target.value;
    setValue(q);
    setOpen(true);
    setActiveIdx(-1);
    schedule(q);
  }

  function commitSelection(label: string) {
    setValue(label);
    setOpen(false);
    const rec = [label, ...loadRecents().filter((x) => x !== label)];
    saveRecents(rec);
    setAnnounce(`Selected ${label}`);
  }

  function onKeyDown(e: KeyboardEvent<HTMLInputElement>) {
    if (!open) return;
    if (e.key === 'ArrowDown') {
      e.preventDefault();
      setActiveIdx((i) => Math.min(items.length - 1, i + 1));
    } else if (e.key === 'ArrowUp') {
      e.preventDefault();
      setActiveIdx((i) => Math.max(0, i - 1));
    } else if (e.key === 'Enter' && activeIdx >= 0) {
      e.preventDefault();
      commitSelection(items[activeIdx].label);
    } else if (e.key === 'Escape') {
      setOpen(false);
    }
  }

  return (
    <div className="typeahead">
      <label htmlFor="typeahead-input">Search</label>
      <input
        id="typeahead-input"
        role="combobox"
        aria-expanded={open}
        aria-controls={listId}
        aria-activedescendant={activeIdx >= 0 ? `${listId}-opt-${items[activeIdx].id}` : undefined}
        aria-autocomplete="list"
        value={value}
        onChange={onChange}
        onKeyDown={onKeyDown}
        onFocus={() => {
          setOpen(true);
          if (!value) runQuery('');
        }}
        onBlur={() => setTimeout(() => setOpen(false), 120)}
      />
      <div id={statusId} role="status" aria-live="polite" className="sr-only">
        {announce}
        {loading ? ' Loading.' : ''}
      </div>
      {open && (
        <ul id={listId} role="listbox" className="typeahead-list">
          {items.map((s, idx) => (
            <li
              key={s.id}
              id={`${listId}-opt-${s.id}`}
              role="option"
              aria-selected={idx === activeIdx}
              onMouseDown={(ev) => ev.preventDefault()}
              onClick={() => commitSelection(s.label)}
            >
              {highlight(s.label, value)}
            </li>
          ))}
        </ul>
      )}
    </div>
  );
}

Recent searches and trending

Recents are client-persisted strings shown when the query is empty. Trending should be fetched once (or edge-cached) as a static JSON blob to avoid coupling hot paths to OLTP.

Rate limiting

  • Client-side — debounce + max concurrent requests (1) + exponential backoff on 429; jitter to prevent synchronized retries.
  • Server-side — token bucket per IP + per API key; separate budgets for authenticated users; optionally edge rate limits on CDN.
// client backoff sketch
async function fetchWithBackoff(url: string, init: RequestInit, attempt = 0): Promise<Response> {
  const res = await fetch(url, init);
  if (res.status === 429 && attempt < 4) {
    const ra = Number(res.headers.get('Retry-After')) || 2 ** attempt;
    await new Promise((r) => setTimeout(r, ra * 1000 + Math.random() * 250));
    return fetchWithBackoff(url, init, attempt + 1);
  }
  return res;
}

Deep-dive Q&A

Why blur + setTimeout for closing the list? Senior

Clicking an option fires blur on the input before click handlers run. A short timeout lets mousedown preventDefault on the option preserve selection; alternatively use focusin/focusout with relatedTarget checks.

How do you handle multilingual tokenization? L6

Highlighting on raw substring breaks for composed scripts and diacritics. Prefer server-returned match ranges or ICU-based normalization on both ends; never rely on naive toLowerCase() for Turkish “I” issues in global products.

Should suggestions be virtualized? Arch

For <200 items, DOM lists are fine. For huge catalogs, virtualize and ensure aria-activedescendant still points to stable IDs; test with screen readers.
