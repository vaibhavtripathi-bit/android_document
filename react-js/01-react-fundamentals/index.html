Series 1: React Fundamentals

From JSX compilation to fiber architecture — the internal mechanics every Staff-level frontend engineer must master for Google L6 interviews.

This series deconstructs React from the ground up at the depth expected for a Google L6 / Staff-level interview. We start with how JSX transforms into plain JavaScript objects, trace the component model's evolution, explore the fiber architecture that powers concurrent rendering, and examine the reconciliation algorithm that makes virtual DOM diffing efficient. Each document focuses on internals that distinguish a Staff engineer from a Senior one.

Documents in This Series

01 — JSX Transformation

JSX is not HTML. Babel/SWC transformation pipeline, React.createElement vs automatic JSX runtime (jsx, jsxs, jsxDEV), pragma configuration, Fragment desugaring, and development vs production transforms.

02 — Component Model

Function vs class components internal representation, ReactElement type structure, $$typeof Symbol for XSS protection, children normalization, component identity, React.lazy/Suspense, and the hooks evolution.

03 — Fiber Architecture

Fiber node structure, current vs workInProgress double-buffering, linked-list traversal, unit of work concept, requestIdleCallback origin story, incremental rendering, and the priority lanes system.

04 — Virtual DOM & Diffing

Reconciliation algorithm: O(n) heuristic vs O(n³) tree edit distance, key assumptions, same-type reuse, cross-type destruction, list diffing with keys, and why index-as-key is problematic.

05 — Element vs Component vs Instance

ReactElement (immutable description) vs Component (template) vs Fiber (mutable instance). type field semantics, lazy components, React.memo wrapping, forwardRef, createElement vs cloneElement.

06 — Context System

createContext internals, Provider/Consumer propagation through fiber tree, bail-out optimization, why context triggers re-renders, splitting strategies, use(Context) in React 19, and module-level patterns.

07 — Error Boundaries

componentDidCatch vs getDerivedStateFromError timing, error propagation through fiber tree, recovery patterns, reset keys, React 19 createRoot error callbacks, and production error reporting.

08 — Interview Q&A

55+ L6-level interview questions with detailed answers covering JSX compilation, fiber architecture internals, reconciliation, virtual DOM diffing, context performance, error boundaries, and component model evolution.
