Architect-Level Focus Areas

Throughout the documentation, orange Architect callouts highlight knowledge that distinguishes a Staff/L6 engineer from a senior engineer. Key areas:

  • Fiber architecture — understanding the linked-list tree, double-buffering, and incremental rendering (Series 1)
  • Scheduler & lanes — priority-based rendering, time slicing, starvation prevention (Series 2)
  • Hook dispatcher switching — how React enforces rules of hooks at runtime (Series 3)
  • State library internals — Redux middleware pipeline, Zustand subscription model, signals reactivity (Series 4)
  • Data loading architecture — route-level loaders, streaming, race condition handling (Series 5)
  • Core Web Vitals — measuring and optimizing LCP, INP, CLS in React apps (Series 6)
  • RSC wire format — Flight protocol, server/client boundary serialization (Series 7)
  • Testing at scale — MSW architecture, parallel E2E, performance regression testing (Series 8)
  • Bundler internals — Webpack/Vite/Turbopack architecture, HMR, tree shaking (Series 9)
  • Browser security model — CSP, CORS, same-origin policy, Trusted Types (Series 10)
  • Micro-frontend architecture — Module Federation, shared dependencies, team autonomy (Series 11)
  • V8 JIT pipeline — hidden classes, inline caches, deoptimization triggers (Series 12)
